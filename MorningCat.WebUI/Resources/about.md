# MorningCat
MorningCat 是一个基于 NapCat 的 QQ 机器人框架
*新版变成OneBot协议了*
## 技术栈
- **后端**：.NET 10, ASP.NET Core
- **前端**：React 19, TypeScript, HeroUI
- **通信**：WebSocket（OneBot）
 *我自己的WebUi太烂所以改的NapCat的样式设计，不要喷我喵QAQ*
