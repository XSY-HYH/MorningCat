# MorningCat
MorningCat 是一个基于 NapCat 的 QQ 机器人框架，支持模块化插件开发。
## 特性
- **模块化设计**：支持动态加载、卸载插件
- **WebUI 管理**：提供现代化的 Web 管理界面
- **命令系统**：灵活的命令注册和处理机制
- **配置管理**：支持 YAML 格式的配置文件
## 技术栈
- **后端**：.NET 10, ASP.NET Core
- **前端**：React 19, TypeScript, HeroUI
- **通信**：WebSocket (NapCat)
 *WebUi借鉴NapCat的WebUi*
